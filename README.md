# MVS

## Setup Instructions

1. Clone the repository
2. Navigate to the frontend directory:
   ```bash
   cd frontend/my-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
   or
   ```bash
   yarn start
   ```

## Dependencies

- React
- React Router DOM
- Axios
